# Детальная информация о маршрутах

Этот документ описывает структуру детальных данных о маршрутах общественного транспорта Рыбинска.

## Структура файлов

Детальная информация о каждом маршруте хранится в отдельных JSON файлах в папке `api/route-details/`:

- `bus-1.json` - Автобусный маршрут № 1
- `trolleybus-1.json` - Троллейбусный маршрут № 1
- и т.д.

## Формат данных маршрута

```json
{
  "route_id": "1",
  "route_number": "1",
  "route_name": "ПАТП № 1 - Улица Буксирная",
  "type": "bus",
  "status": "active",
  "directions": {
    "forward": {
      "name": "ПАТП № 1 → Улица Буксирная",
      "stops_count": 26,
      "length_km": 14.80,
      "duration_minutes": 45,
      "stops": ["ПАТП № 1", "..."]
    },
    "backward": {
      "name": "Улица Буксирная → ПАТП № 1",
      "stops_count": 24,
      "length_km": 15.20,
      "duration_minutes": 48,
      "stops": ["Улица Буксирная", "..."]
    }
  },
  "schedules": {
    "weekday": {
      "forward": {
        "departure_point": "ПАТП № 1",
        "times": ["05:05", "05:25", "..."]
      },
      "backward": {
        "departure_point": "Улица Буксирная",
        "times": ["05:52", "06:12", "..."]
      }
    },
    "saturday": {...},
    "sunday": {...},
    "weekend": {...}
  }
}
```

## Типы расписаний

- `weekday` - Рабочие дни (понедельник - пятница)
- `saturday` - Суббота
- `sunday` - Воскресенье
- `weekend` - Выходные дни (суббота и воскресенье)
- `winter` - Зимний период (16 октября - 14 апреля)
- `summer` - Летний период (15 апреля - 15 октября)
- `special` - Специальные дни (праздники, особые даты)

## Использование

Для получения детальной информации о маршруте:

```javascript
// Получить информацию о маршруте № 1
fetch('api/route-details/bus-1.json')
  .then(res => res.json())
  .then(data => {
    console.log('Остановки прямого направления:', data.directions.forward.stops);
    console.log('Расписание в рабочие дни:', data.schedules.weekday.forward.times);
  });
```

## Примечания

- Все времена указаны в формате HH:MM (24-часовой формат)
- Протяженность указана в километрах
- Время движения указано в минутах
- Некоторые маршруты имеют разные расписания для летнего и зимнего периодов
- Некоторые маршруты работают только в определенные дни или периоды
