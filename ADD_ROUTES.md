# Инструкция по добавлению маршрутов

Для добавления детальной информации о маршрутах создайте файлы в формате `api/route-details/{type}-{number}.json`.

## Формат имени файла

- Автобусные маршруты: `bus-{номер}.json` (например, `bus-1.json`, `bus-2.json`)
- Троллейбусные маршруты: `trolleybus-{номер}.json` (например, `trolleybus-1.json`)
- Маршруты с буквами: `bus-8T.json`, `bus-10S.json`, `bus-2N.json` и т.д.

## Структура JSON файла

```json
{
  "route_id": "номер_маршрута",
  "route_number": "номер_для_отображения",
  "route_name": "Название маршрута",
  "type": "bus" | "trolleybus",
  "status": "active" | "planned",
  "planned_from": "2026-01-01" (только для planned),
  "directions": {
    "forward": {
      "name": "Начало → Конец",
      "stops_count": число,
      "length_km": число,
      "duration_minutes": число,
      "stops": ["Остановка 1", "Остановка 2", ...]
    },
    "backward": {
      "name": "Конец → Начало",
      "stops_count": число,
      "length_km": число,
      "duration_minutes": число,
      "stops": ["Остановка 1", "Остановка 2", ...]
    }
  },
  "schedules": {
    "weekday": {
      "forward": {
        "departure_point": "Название начальной остановки",
        "times": ["05:30", "05:44", ...]
      },
      "backward": {
        "departure_point": "Название конечной остановки",
        "times": ["06:08", "06:22", ...]
      }
    },
    "saturday": {...},
    "sunday": {...},
    "weekend": {...},
    "winter": {...},
    "summer": {...}
  }
}
```

## Примеры

См. файлы:
- `api/route-details/bus-1.json` - пример автобусного маршрута
- `api/route-details/bus-3.json` - еще один пример
- `api/route-details/trolleybus-1.json` - пример троллейбусного маршрута

## Примечания

1. Все времена в формате `HH:MM` (24-часовой формат)
2. Протяженность в километрах (число с точкой)
3. Время движения в минутах (целое число)
4. Список остановок должен точно соответствовать порядку движения
5. Для маршрутов с разными расписаниями в разные периоды используйте ключи `winter` и `summer`
6. Для специальных дней используйте ключ `special` с указанием дат

## Автоматизация

Для массового добавления маршрутов можно использовать скрипт на Python или JavaScript, который будет парсить исходные данные и генерировать JSON файлы.
